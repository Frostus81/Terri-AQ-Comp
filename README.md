Steuerung für Terrarium und Aquarium über einen Arduino Uno mit TFT-Display von Nextion.

Features:

- Lichtsteuerung
   - Sonnenaufgang
   - Sonnenuntergang
   - Tageslicht
   - Mondlicht
   
- Temperatur- / Luftfeuchtigkeitssteuerung
   - Heizer / Lüftersteuerung
   
- Zeitschaltuhr
   - Lichtstatus via RTC
   - Relais ein / aus
